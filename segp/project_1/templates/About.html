About

This program allows user to input multiple keywords or phrases, which will then return the user with suggestions of relavant current topics that will have an impact on the community based on the program's calculation. Apart from the topic suggestions, the program will return the number of citations and published articles of the input keywords. A graph which displays the score comparison among the keywords will also be shown so that the user can choose with ease.



How To

To use this program, the user shall simply put in a topic/keyword that is related to the area that the user is interested in studying (up to 5 seperate topic). The user is also able to "like" certain topics which will prompt the program to give results more related to the user's public interests.
