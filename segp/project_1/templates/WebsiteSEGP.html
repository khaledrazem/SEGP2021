HOW TO

There are two ways for a user to begin searching for their queries:-

Case 1 allows a user to enter as many queries to the system whereby Case 2 lets a user choose a category followed by their preferences to evaluate the scores (Multiple preferences allowed>

Both Case 1 and Case 2 will then return the user with relevant information paired with their own individual scores. The user can then pick which suggested topic they are most interest in and the system shall return the user a more detailed information regarding the selected topic.
