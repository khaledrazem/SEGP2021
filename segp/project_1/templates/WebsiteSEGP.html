ABOUT US

The purpose of this website is to enable researchers from all fields of study to find and consider relevant topics to be researched before choosing. The user shall simply present their topic to the website before proceeding to submit their requests. The importance of a topic or a combination of topics is determined by the score calculated by certain algorithms according to the user's preference.

HOW TO

There are two ways for a user to begin searching for their queries:-

Manual search allows a user to enter as many queries to the system whereby Field search lets a user choose a category followed by their preferences to evaluate the scores (Multiple preferences allowed)

Both Manual search and Field search will then return the user with relevant information paired with their own individual scores. The user can then pick which suggested topic they are most interest in and the system shall return the user a more detailed information regarding the selected topic.
